# Figma 광고 레이아웃 레퍼런스

> B2G-광고-상페 파일 (node-id: 1179:3) — 실제 집행된 3가지 광고 레이아웃 패턴 분석 결과

---

## 패턴 A — 헤드라인밴드형

**레이아웃 구조**
- 배경: 크림/베이지 상단(#F0E8D5) + 노란 앰버 하단(#FFB300→#FF9500) — warm 밝은 톤
- 헤드라인 영역: 검정(#000) 가로 밴드 2개 (full-width) → 밴드 안에 시안/네온(#00E5FF) 초대형 볼드 텍스트
- 구조: [브랜드 태그 상단] → [밴드1: 헤드라인 1줄] → [밴드2: 헤드라인 2줄] → [중앙 비주얼+훅] → [검정 하단 바+CTA]
- 우측: 수치 카드 (반투명 라운드 박스, 검정 숫자)
- 하단: 검정 바 + 흰색 CTA 텍스트 + 시안 뱃지

**색상**
- 배경: #F0E8D5 크림 → #FFBE00 앰버
- 헤드라인 밴드: #000000 검정
- 헤드라인 텍스트: #00E5FF 시안/네온
- 수치 카드: rgba(0,0,0,0.08) 반투명
- 하단 바: #000000

**타이포그래피**
- 헤드라인: ExtraBold/Black 96px, 10자 이내/줄
- 훅: Bold 34px, 여러 줄 가능
- CTA: Bold 28px 흰색

**적합 앵글**: 혜택형, 가격 임팩트, 수치 강조

## 패턴 B — 다크스플릿형

**레이아웃 구조**
- 상단 55%: 초록 그라디언트(#00C853→#001500) + 코드/UI 스크린샷 반투명 오버레이
- 좌상단: 제품/앱 아이콘 (흰 배경 + 라운드 코너, 110px×110px)
- 경계 구역: 노란 하이라이트 바(#F5FF00 배경 + 검정 볼드 텍스트) — USP 핵심 문구 10자 이내
- 하단 45%: 순검정(#000000) + 흰색 헤드라인 2줄
- 우하단: 초록/노랑 CTA 텍스트 + "→"

**색상**
- 배경 상단: #00C853 → #001500 초록 그라디언트
- 배경 하단: #000000 순검정
- USP 하이라이트: #F5FF00 노란, 검정 텍스트
- 헤드라인: #FFFFFF 흰색
- CTA: #00E676 초록 또는 #F5FF00 노란

**타이포그래피**
- USP 하이라이트: Black 44px, 10자 이내
- 헤드라인: Bold 52px, 2줄 15자 이내
- CTA: Medium 26px

**적합 앵글**: 공감형, 앱/기술 서비스, 신뢰 구축

## 패턴 C — 이미지모자이크형

**레이아웃 구조**
- 배경: 다크 네이비(#1A1A2E) + 도트 그리드 오버레이 패턴
- 상단 35%: 텍스트 영역
  - 흰색 볼드 헤드라인 2줄 (15자 이내/줄)
  - 핫핑크(#FF00FF) USP 뱃지 2개 — pill 형태, 흰 텍스트
- 하단 65%: 2×3 이미지 그리드 (6개 타일)
  - 각 타일: 제품·모델·컨셉 이미지 + 캡션 오버레이

**색상**
- 배경: #1A1A2E 다크 네이비
- 헤드라인: #FFFFFF 흰색
- USP 뱃지 배경: #FF00FF 핫핑크/마젠타
- 뱃지 텍스트: #FFFFFF 흰색
- 이미지 캡션: 흰색 on 반투명 다크 오버레이

**타이포그래피**
- 헤드라인: Black 72px, 2줄
- USP 뱃지: Bold 28px

**적합 앵글**: 긴박형, 시즌 프로모션, 비주얼 어필, 다양한 사례 강조
