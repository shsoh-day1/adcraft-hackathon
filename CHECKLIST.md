# 주예진 콘텐츠 기준 — 광고 소재 체크리스트

> 성과 소재(CTR 기준) 분석을 바탕으로 정리한 필수 기준입니다.
> 광고 소재 생성 시 아래 모든 `[MUST]` 항목을 충족해야 합니다.

---

## 1. 카피 작성 원칙

| ID | 구분 | 항목 | 기준 예시 |
|----|------|------|-----------|
| C1 | MUST | 숫자로 혜택을 명시 | "90% 할인", "4만원대", "선착순 30명" |
| C2 | MUST | 긴박감·희소성 트리거 포함 | "마감 임박", "잠시만", "지금 해야", "선착순 마감" |
| C3 | MUST | 헤드라인은 짧고 임팩트 있는 문장 | "못 참지", "제일 싸요", "수익화까지 가능하다고?" |
| C4 | MUST | 행동 유도 문구(CTA) 명시 | "지금 신청하기", "수익화하기", "배워보세요" |
| C5 | MUST | 의문문·반문으로 호기심 자극 | "할 사람?", "궁금하다면?" — 독자가 스스로 대답하게 만드는 구조 |
| C6 | MUST | 타겟의 상황·고통을 먼저 언급 | "코딩 1도 모르던 문과생" — 공감 후 해결책 제시 순서 |
| C7 | GOOD | 브랜드 톤앤매너 유지 | 구어체·친근한 반말투, 딱딱한 문어체 금지 |

## 2. 비주얼 레이아웃 패턴

| ID | 구분 | 항목 | 기준 예시 |
|----|------|------|-----------|
| V1 | MUST | 헤드라인 폰트가 압도적으로 클 것 | 배경 이미지보다 텍스트가 시각적 우선순위 |
| V2 | MUST | 핵심 숫자·키워드 색상/크기로 강조 | "AI", "90%" — 색상 분리 또는 크기 차이 |
| V3 | MUST | 배경-텍스트 가독성 확보 | 어두운 배경 + 흰 텍스트 / 밝은 배경 + 진한 텍스트 |
| V4 | MUST | 메인 카피 최대 2줄, 줄당 20자 이내 | "코딩 몰라도 웹서비스 만들 수 있어?" (19자) |
| V5 | MUST | 서브 카피 최대 2줄, 줄당 20자 이내 | "이번 주 선착순 50명만 4만원대." (18자) |
| V6 | MUST | 시선 흐름 순서 준수 | 헤드라인 → 핵심 숫자 → 서브카피 → CTA 순으로 배치 |
| V7 | MUST | 헤드라인이 서브카피보다 1.5배 이상 크게 | 크기 차이로 시각적 계층 명확히 구분 |
| V8 | MUST | CTA는 하단 고정 | 버튼·행동 유도 문구는 항상 소재 하단에 위치 *(UI 렌더링 처리)* |
| V9 | MUST | 텍스트 최소 여백 확보 | 텍스트가 이미지 경계에서 최소 5% 이상 띄울 것 *(UI 렌더링 처리)* |
| V10 | MUST | 플랫폼별 안전 영역 준수 | 상하 15% 이내 텍스트 금지 / 9:16·1:1 레이아웃 분리 적용 *(UI 렌더링 처리)* |
| V11 | GOOD | 캐릭터·말풍선·SNS 캡처 등 친근한 포맷 | 토끼 캐릭터, 말풍선, 트위터 형태 — 광고처럼 안 보이게 |
| V12 | GOOD | 인물 또는 제품 이미지 포함 | 실제 인물 사진, 제품 클로즈업, 캐릭터 이미지 |
| V13 | GOOD | 컬러 포인트 1~2가지로 집중 | 검정+분홍, 검정+형광 — 시선 분산 방지 |
