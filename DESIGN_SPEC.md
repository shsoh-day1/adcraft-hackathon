## 1. 레이아웃 구조

### 기본 3단 구조 (모든 소재 공통)

```
┌─────────────────────────┐
│  [헤드라인]              │  ← 전체 높이의 40~50% 차지
│  짧고 강렬한 1~2줄       │
├─────────────────────────┤
│  [비주얼 영역]           │  ← 인물/제품/캐릭터 이미지 또는 강조 숫자
│  이미지 or 핵심 숫자     │
├─────────────────────────┤
│  [CTA 버튼 + 서브 카피]  │  ← 하단 고정
└─────────────────────────┘
```

### 변형 패턴 (택1)

| 패턴 | 특징 | 적합한 상황 |
|------|------|-------------|
| **숫자 중심형** | 할인율/가격을 화면 중앙에 크게 배치 | 가격 혜택이 핵심일 때 |
| **헤드라인 압도형** | 텍스트가 전체 면적 60% 이상 | 메시지 강도가 핵심일 때 |
| **SNS 캡처형** | 실제 SNS 게시물처럼 구성 | 신뢰감·소셜 증명이 핵심일 때 |
| **캐릭터 대화형** | 말풍선 + 캐릭터 이미지 | 친근함·공감이 핵심일 때 |

## 2. 타이포그래피

### 헤드라인
- **폰트**: 고딕 계열 (Noto Sans KR Black, Pretendard ExtraBold 등)
- **크기**: 전체 소재 너비 대비 글자 1개가 10~15% 차지하는 비율
- **굵기**: ExtraBold / Black (900) 이상
- **색상**: 흰색(`#FFFFFF`) 또는 형광 노랑(`#FFE600`) — 어두운 배경 위
- **줄 수**: 최대 3줄, 줄당 6자 이내 권장
- **줄간격**: 타이트하게 (line-height 1.1~1.2)

### 강조 키워드 (헤드라인 내 1~2개)
- 다른 색으로 분리: 분홍(`#FF4FA3`), 형광 민트(`#00FFD1`), 형광 노랑(`#FFE600`)
- 또는 크기를 1.3~1.5배로 키워 강조

### 서브 카피
- 크기: 헤드라인의 30~40% 수준
- 색상: 흰색 또는 밝은 회색(`#CCCCCC`)
- 역할: 헤드라인 보조 설명, 타겟 공감 문장

### CTA 버튼 텍스트
- 크기: 서브 카피와 동일하거나 약간 크게
- 배경: 밝은 색 버튼 (흰색, 형광, 밝은 핑크)
- 텍스트 색: 버튼 배경과 대비되는 진한 색

## 3. 색상 시스템

### 배경색 (어두운 계열 필수)

| 용도 | 색상 | 헥스 코드 |
|------|------|-----------|
| 기본 배경 1 | 검정 | `#0A0A0A` ~ `#1A1A1A` |
| 기본 배경 2 | 진한 네이비 | `#0D0D2B` ~ `#1A1A3E` |
| 기본 배경 3 | 진한 보라 | `#1A0A2E` |

> ⚠️ 흰색/밝은 배경은 사용하지 않음. 소재가 피드에서 묻혀버림.

### 포인트 컬러 (1~2가지만 선택)

| 조합 | 포인트 1 | 포인트 2 | 사용 소재 패턴 |
|------|----------|----------|----------------|
| **핑크+형광** | `#FF4FA3` | `#FFE600` | 할인 강조형 |
| **민트+흰색** | `#00FFD1` | `#FFFFFF` | 정보 전달형 |
| **형광노랑+흰색** | `#FFE600` | `#FFFFFF` | 긴박감 강조형 |

### 강조 숫자/배지 처리
- 숫자를 별도 색상 블록(쿠폰 형태)으로 분리
- 예: 흰 배경 박스 안에 핑크 텍스트로 "90%"
- 또는 형광 배경에 검정 텍스트

## 4. 비주얼 요소 배치

### 인물/제품 이미지
- 배경과 자연스럽게 합성 (누끼 처리)
- 텍스트와 겹쳐도 되지만, 헤드라인 가독성 우선
- 위치: 우측 또는 중앙 하단

### 캐릭터/말풍선 패턴
- 캐릭터는 화면 우측 하단 배치
- 말풍선은 캐릭터 위쪽에 위치
- 말풍선 내 텍스트: 일상 구어체, 작은 폰트

### SNS 캡처 패턴
- 실제 트위터/스레드 UI 그대로 재현
- 프로필 사진 + 닉네임 + 날짜 + 조회수 포함
- 배경은 어둡게 처리해서 캡처가 떠보이게

### 돈/성공 상징 요소
- 달러 이모지, 금화 이미지 등 소량 배치
- 과하면 스팸처럼 보임 — 3개 이내

## 5. 소재 사이즈별 레이아웃 조정

| 플랫폼 | 비율 | 헤드라인 위치 | 특이사항 |
|--------|------|---------------|----------|
| 인스타그램 피드 | 1:1 (1080×1080) | 상단 1/3 | 텍스트 중앙 정렬 |
| 인스타그램 스토리 | 9:16 (1080×1920) | 상단 30% | 하단 30%는 CTA 전용 |
| 페이스북 피드 | 1.91:1 (1200×628) | 좌측 배치 | 이미지는 우측 |
| 유튜브 썸네일 | 16:9 (1280×720) | 좌측 또는 중앙 | 얼굴 이미지 우측 필수 |

## 6. 절대 하지 말 것 (Anti-pattern)

- ❌ 밝은 배경 (흰색, 베이지, 연회색) — 피드에서 묻힘
- ❌ 폰트 3가지 이상 혼용
- ❌ 헤드라인 4줄 이상
- ❌ 포인트 컬러 3가지 이상
- ❌ 텍스트 없는 순수 이미지형 소재
- ❌ 영문 위주 카피 (한글 구어체가 CTR 높음)
- ❌ 정중한 존댓말 헤드라인 ("~하실 수 있습니다")
